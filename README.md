# MCF Overlay

Broadcast graphics system untuk Mandar Culture Festival.

## URL Produksi

```txt
https://radio-sbl-overlay.web.app/control
https://radio-sbl-overlay.web.app/overlay/main
https://radio-sbl-overlay.web.app/overlay/left
https://radio-sbl-overlay.web.app/overlay/right
https://radio-sbl-overlay.web.app/preview
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Firebase

Salin `.env.example` ke `.env`, lalu isi konfigurasi Firebase. File `.env` tidak ikut Git.
